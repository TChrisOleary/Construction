# Construction Company Website (WordPress + Elementor)

## Overview
A modern, responsive construction company website built using WordPress and Elementor. Designed with a strong visual hierarchy, clear service sections, and a professional, trustworthy layout.

## Features
- Custom homepage with hero section
- Services overview with icons and structured layout
- About/Company section
- Contact section with form integration
- Fully mobile-responsive design
- Elementor-based visual editing

## Tools Used
- WordPress
- Elementor
- Canva (graphics)
- MAMP (local development)

## My Role
Designed and built the full site layout, including structure, styling, and responsive behavior. Focused on clarity, readability, and a clean contractor aesthetic.
